Ejemplo

    1. Preparación de Datos y Clave:

        Mensaje original: "Hola Mundo!"
        Clave de cifrado: "ClaveDES123"
        El mensaje se organiza en un bloque de datos (64 bits) y la clave se procesa para su uso en las rondas de cifrado.

    2. Rondas de Cifrado:

        DES utiliza 16 rondas de cifrado. Cada ronda consiste en varios pasos:
        
            1. Expansión: Se expanden los bits de datos para prepararlos para la mezcla con la clave.
            2. Mezcla con la Subclave: Se combina con una subclave generada de la clave principal.
            3. Sustitución (S-Box): Los bits expandidos se reemplazan según tablas de sustitución.
            4. Permutación: Se reorganizan los bits para prepararlos para la siguiente ronda.

    3. Resultado del Cifrado:

        El proceso de cifrado transforma el texto original en un texto cifrado completamente diferente. 
        Texto cifrado: "057744eee2602362f0654b783ed99bd56c0a97a634a20af0"
